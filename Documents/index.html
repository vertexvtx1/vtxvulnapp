Server FAQ

Do I need VPS or a Dedicated server to run this system?

No, You can run this system in a shared host as well and it will run smoothly. But if you have much traffic, you may need to upgrade your server/hosting. Our codes are highly optimized to ensure the best possible performance.

You can choose any hosting provider that provides cPanel-based hosting. cPanel is not a must, but we recommend you to have cPanel-based hosting.

I don't have cPanel, What should I do?

Nothing to worry about, Our system will work with any control panel but cPanel is easy to manage, for that we recommend cPanel-based hosting. Also, Our free Support is limited to cPanel-based hosting only.

Do you provide hosting as well?

Admin Dashboard
Dashboard Overview

From the Dashboard menu, you will get an overview at a glance. You may know the total selling amount, total purchase amount, total purchase return amount, total sales return amount, and many others.

Manage Product
Manage Product Overview

In the Manage Product menu, there are 4 submenu: Categories, Brands, Unit & Products.

Categories
From Category menu admin can see the list of categories. To add a new category there is an Add New button. Admin can update the category name by click on the Edit button at the right side of each category. A category can be removed permanently if no product added with that category. Admin can also search a specific category by its name.
Brands
From Brand menu admin can see the list of brands. To add a new brand there is an Add New button. Admin can update the brand name by click on the Edit button at the right side of each brand. A brand can be removed permanently if no product added with that brand. Admin can also search a specific brand by its name.
Units
From Unit menu admin can see the list of unit. To add a new unit there is an Add New button. Admin can update the unit name by click on the Edit button at the right side of each unit. An unit can be removed permanently if no product added with that unit. Admin can also search a specific unit by its name.
Products
From Products menu admin can see the list of products. To add a new product there is an Add New button. Admin can update the product name by click on the Edit button at the right side of each product. Admin can search a specific product by its name. Admin can download product list by click on PDF button at the Top-Right corner.
Warehouse
Warehouse Overview
From Warehouse menu admin can see the warehouse list. To add a new warehouse there is an Add New button. Admin can update the warehouse name by click on the Edit button at the right side of each warehouse. A warehouse can be disabled from Edit menu.
Staff
Staff Overview

From the Staff menu, admin can see the staff list. Staffs have some specific access from the whole system. To add a new staff there is an Add New button. Admin can update the information of a staff by click on the Edit button at the right side of each staff. Super admin can also login as a staff.

Customer
Customer Overview

From the Customer menu, admin can see the customer list. To add a new customer there is an Add New button. Admin can update the information of a customer by click on the Edit button at the right side of each customer.

Admin can send notification to all customer by email/sms by Notification All button. Also admin is able to send notification to a specific customer from notify button.

All unsettled payment (Receivable & Payable) with a customer can be clear from the Payment menu. If all payments is cleared then the payment button will be disabled.

Supplier
Supplier Overview

From the Suppler menu, admin can see the supplier list. To add a new supplier there is an Add New button. Admin can update the information of a supplier by click on the Edit button at the right side of each supplier.

All unsettled payment (Receivable & Payable) with a supplier can be clear from the Payment menu. If all payments is cleared then the payment button will be disabled.

Purchase
Purchase Overview

From the All Purchase menu, admin can see the purchase list. To add a new purchase there is a Add New button. Admin can update a purchase data by click on the Edit button at the right side of each purchase. If a purchase has some returns then that purchase can't be updated.

The red circle () besides invoice no. denotes that some products of that purchase has returned. Admin can search a purchase data by the invoice number or supplier name or supplier mobile number. Also admin can search purchase data by date to date.

More Button:
In More button you will get options:
* To pay the payable amount or return back the extra payment.
* Return purchased products to supplier.
* Download details as pdf

Purchases Return

From the Purchases Return menu, admin can see the purchase return list, download data as pdf, Edit purchase return data and download the details from more button.

Sale
Sale Overview

From the All Sale menu, admin can see the sales list. To add a new sale there is a Add New button. Admin can update a sale data by click on the Edit button at the right side of each sale. If a sale has some returns then that sale can't be updated.

The red circle () besides invoice no. denotes that some products of that sale has returned. Admin can search a sale data by the invoice number or customer name or customer mobile number. Also admin can search sale data by date to date.

More Button
In More button you will get options:
* To receive the receivable amount or return back the extra payment.
* Return sold products to supplier.
* Download details as pdf

Sales Return

From the Sales Return menu, admin can see the sale return list, download data as pdf, Edit sale return data and download the details from more button.

Adjustment
Adjustment Overview

From the Adjustment menu, admin can see the list of adjustment. Admin can add new products or subtract products from stocked data. For a new adjustment there is an Add New button. Admin can update adjustment data by click on the Edit button at the right side of each adjustment.

Transfer
Transfer Overview

From the Transfer menu, admin can see the list of transfers. Admin can transfer products from one warehouse to another warehouse. For a new transfer there is an Add New button. Admin can update transfer data by click on the Edit button at the right side of each transfer.

Expense
Expense Overview
Type

From the Type menu, admin can see the list of expense types. To add a new Expense Type there is an Add New button. Admin can update expense Type data by click on the Edit button at the right side of each expense type.

All Expense

From the All Expense menu, admin can see the list of expenses. To add a new Expense there is an Add New button. Admin can update expense data by click on the Edit button at the right side of each expense type.
